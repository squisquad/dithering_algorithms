## Dithering algorithms
Simple Processing implementations of some dithering algorithms
	
* Ordered dither
* Floyd-Steinberg
* Atkinson
* Random

Monochrome and 3bit RGB version

_Author [@danielepiccone](https://twitter.com/danielepiccone) - built with Processing 2.0b8_
